## Setup

```sh
git clone https://github.com/GPT-Engineer-App/gatsby-pdf-figma-website.git
cd gatsby-pdf-figma-website
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
